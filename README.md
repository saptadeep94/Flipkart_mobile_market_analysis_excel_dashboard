# Flipkart Mobile Market Analysis Dashboard

## Project Overview

This project is an interactive Excel dashboard designed to analyze Flipkart mobile market data and generate business insights related to brand performance, customer satisfaction, pricing strategy, profitability, and value-for-money trends.

## Objectives

* Analyze top-performing mobile brands and models
* Evaluate customer satisfaction using ratings
* Compare Budget, Mid-range, and Premium segments
* Perform profitability and margin analysis
* Measure Value-for-Money (VFM) performance

## Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* KPI Cards
* Interactive Slicers
* Data Segmentation
* Business Reporting

## Key Insights

* Mid-range smartphones provide the strongest balance between pricing and customer satisfaction.
* Premium brands maintain higher margins but lower VFM performance.
* Product pricing shows weak correlation with customer ratings.
* Market performance is concentrated among a few dominant brands.

## Dashboard Features

* Interactive filters using slicers
* Brand and model-level analysis
* Segment analysis
* KPI-driven dashboard design
* Business recommendations and reporting

## Dataset Limitations

* Dataset does not include date fields, limiting YoY and MoM trend analysis.
* Customer demographic and sales quantity data were unavailable.
